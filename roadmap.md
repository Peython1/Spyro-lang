# Roadmap (Spyro)

Lista de ideias e próximos passos.

## Curto prazo
- Melhorar mensagens de erro (linha/coluna, contexto).
- Mais testes de parser e semântica.
- Padronizar módulos e import.

## Médio prazo
- Bytecode mais estável e documentado.
- Otimizações simples (const folding, cache de lookup).
- Melhorar interoperabilidade com Python (FFI controlada).

## Longo prazo
- VM dedicada (ou JIT).
- Ferramentas: formatter, linter, language server (LSP).
- Package manager para módulos Spyro.
